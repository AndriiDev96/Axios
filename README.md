## Installing

  1) Using npm: 

```bash
$ npm install 
```

  2) Create directory **dist**

```bash
$ gulp directories
```

  3) Move folder **framework** to directory **dist**

  4) Initialize files

```bash
$ gulp deploy
```

  5) Run server

```bash
$ gulp
```
